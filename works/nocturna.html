Nocturna Ediciones is a independent publishing company based in Madrid. They are committed to publishing quality fiction for young readers and adults.

December, 2014

The Challange

Although Nocturna already had a website, they weren’t satisfied with their own CMS. It was not filling their needs, and it also had usability and look-and feel issues. As a result, publishing workflow was unorganized and often chaotic. Furthermore, their CMS used deprecated PHP code and an unormalized MySQL database.

Normalising Database and Code

Deprecated PHP code was removed or updated to 5.5 version. Some functions were overwritten in order to improve the data management workflow. Furthermore, the database was normalised to avoid any data inconsistencies and provide faster data access. For instance, the database did’t have any junction tables at all, so tables were not properly related.

Restructuring Workflow Process to Improve Publishing Efficiency

A new responsive template was implemented and customized to meet Nocturna’s needs. Moreover, workflow was restructured in order to improve usability, while increasing the effectiveness and efficiency of publishing process.
